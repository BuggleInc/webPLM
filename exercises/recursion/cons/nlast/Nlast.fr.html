Nlast

Given a [!scala]List[Int][/!][!java|python]recursive list[/!] and an integer N, return the list with only the N last elements.

Votre solution devrait être linéaire en temps, mais vous aurez probablement besoin de définir plusieurs fonctions pour atteindre cette complexité.

L'astuce est de calculer une seule fois (et à l'avance) combien d'éléments il faut garder.
