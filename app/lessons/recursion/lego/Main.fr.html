Algorithmes récursifs

Cette leçon vous permet d'expérimenter avec des algorithmes récursifs.

Si vous cherchez d'autres algorithmes récursifs, un exercice sur les tris récursifs (en particulier QuickSort et MergeSort) est prévu à l'avenir dans la leçon sur les tris.

Que puis-je faire pour améliorer cette leçon ?

Comme souvent, plusieurs points pourraient être améliorés dans cette leçon :

  • Le triangle de Sierpinski peut être tracé de façon bien plus simple en utilisant un L-système. Voir http://jpvallon.free.fr/recursivite.html
  • la courbe de Sierpinski est également intéressante: http://www.mathcurve.com/fractals/sierpinski/sierpinskitriangle.shtml
  • La courbe du dragon pourrait être expliquée comme un L-système. http://ecademy.agnesscott.edu/~lriddle/ifs/heighway/heighway.htm
  • La courbe du dragon2 s'appelle la courbe des dragons jumeaux: http://ecademy.agnesscott.edu/~lriddle/ifs/heighway/twindragon.htm
  • Les courbes remplissant l'espace (Peano, Hildebert, etc) pourraient être ajoutées. http://teachout1.net/village/fill.html
  • Un exercice sur les polygones inscrits serait intéressant: http://www.inrialpes.fr/helix/people/genoud/ENSJAVA/tds/sujets/dessinsReccursifs/actionsrecursives.html
